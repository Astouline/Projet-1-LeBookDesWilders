Le Book des Wilders

Formation développeur web en 5 mois. La Wild Code School te propose une formation développeur web à Reims de 5 mois : formation Javascript / React / Node JS (rentrée septembre et février) ou JavaEE (rentrée octobre 2018). Le campus est situé au sein de Quartier libre, lieu totem numérique dédié aux startups. Viens te former au métier de développeur web au sein d’un écosystème numérique en ébullition : incubateurs comme Innovact, espaces de coworking comme La Capsule, etc.
